# Paper Motion Studio

A browser tool for making stop-motion-style animations from paper-cutout SVG pieces.

## Use it

Open `index.html` in any browser — no build, no dependencies.

## Features

- Drag-and-drop SVG pieces, arrange them on the canvas
- Stop-motion jitter (position / rotation / scale) at a stepped framerate
- Transform prompts: type "transform into an arrow" (or heart, star, circle, letters…) and the pieces travel into that shape and back, looping forever
- Per-piece or multi-select recoloring, scaling, rotation
- Background color or image, custom canvas size
- Export as a self-contained animated SVG
- Save / load projects as JSON

## Publish with GitHub Pages

1. Push these files to a repo
2. Repo Settings → Pages → Source: "Deploy from a branch" → branch `main`, folder `/ (root)`
3. Your tool goes live at `https://<username>.github.io/<repo>/`
